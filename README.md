A premium, portfolio-quality Scientific Calculator web application built entirely with HTML, CSS, and Vanilla JavaScript. It features a modern glassmorphism UI, dark and light modes, basic and scientific operations, and a beautiful landing page.

**[Live Demo](#) | [GitHub Repository](#)**

## Features

- **Beautiful Landing Page**: Smooth scrolling, hero section, features list.
- **Glassmorphism UI**: Premium visual design with soft shadows and background blur.
- **Dark & Light Mode**: Seamless theme switching with animated transitions.
- **Basic & Scientific Operations**: Addition, subtraction, trigonometric functions, logarithms, factorial, etc.
- **Smart Keyboard Support**: Full keyboard accessibility for rapid calculations.
- **Calculation History**: Stores the last 20 calculations locally (persists across reloads).
- **Responsive Design**: Works perfectly on mobile, tablet, and desktop screens.
- **Error Handling**: Graceful handling of invalid expressions and divide-by-zero.

## Installation

This project requires no build tools or package managers. Simply clone the repository and run it locally.

1. Clone the repository:
   ```bash
   git clone [GitHub Repository URL]
   ```
2. Navigate to the project directory:
   ```bash
   cd calculator
   ```
3. Open `index.html` in your preferred web browser.

## Usage

- Scroll down from the landing page or click "Launch Calculator".
- Use the on-screen keypad or your physical keyboard to enter calculations.
- Use the theme toggle button to switch between dark and light modes.
- Open the history panel to view your past calculations.
